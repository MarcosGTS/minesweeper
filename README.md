## Modo de jogo

Clique na celula, o numero que aparecer representa a quantidade de minas escondidas nas celulas adjacentes

Evite todas as minas e ganhe o jogo

**Comandos**
- Clique Esquerdo: Revela celula
- Clique Direito: Marca celula
